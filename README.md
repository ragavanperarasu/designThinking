# 🌞 Autonomous Solar-Powered Rural Monitoring System

## 📌 Project Overview
Many rural areas and village roads lack proper **street lighting, surveillance, and internet connectivity**, leading to safety concerns—especially at night.

To address this issue, we propose an **Autonomous Solar-Powered Rural Monitoring System**. This system combines **renewable energy, smart monitoring, and long-range communication** to enhance rural safety and infrastructure.

---

## 🚀 Key Features

- 🌞 **Solar Powered**
  - Works without electricity connection
  - Eco-friendly and cost-effective

- 📍 **GPS Tracking**
  - Identifies exact location of each street light

- 📡 **LoRa Communication**
  - Long-range data transmission
  - Works without continuous internet

- 🎥 **FPV Camera**
  - Captures real-time video surveillance

- 🧠 **Future AI Integration**
  - Detects suspicious activities
  - Sends alerts to authorities
